Era is an easy-to-medium difficulty machine from Hack The Box that begins with enumerating a web application and uncovering hidden functionality through directory and parameter discovery. Initial access is typically obtained by exploiting a web vulnerability such as insecure file handling or improper input validation, allowing us to gain a foothold on the system.
